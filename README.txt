Консольный файловый tui-пейджер под linux.
Прообразом интерфейса послужил ranger. (во FreeBSD python27-ranger)
Стадия начальной разработки. Задача, серфинг файловой системы с выходом
в любом месте в консоль интерпретатора. Минимально необходимый функционал.
Сохранение возможности перерасти в полноценный файловый менеджер.

make
sudo make install
sudo make uninstall
